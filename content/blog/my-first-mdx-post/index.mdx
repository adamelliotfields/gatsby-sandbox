Here is a JavaScript code block. This uses [`prism-react-renderer`](https://github.com/FormidableLabs/prism-react-renderer)
instead of `gatsby-remark-prismjs`.

```js
/**
 * @param {number} number
 * @returns {string}
 */
const getFizzBuzz = (number) =>
  number % 15 === 0
    ? 'fizzBuzz'
    : number % 3 === 0
    ? 'fizz'
    : number % 5 === 0
    ? 'buzz'
    : number.toString();
```
